# 🛡️ Mock Security Awareness Plan / خطة التوعية الأمنية الافتراضية

**Alignment:** NCA ECC-1:2018 (Control 1-4: Cybersecurity Awareness)  
**Target:** All Enterprise Employees / جميع موظفي المنشأة  

---

## 📌 Overview / نظرة عامة
**English:** This document outlines the security awareness activities conducted to mitigate human-centric risks (Phishing, Social Engineering) and improve the organization's overall security posture.
** توضح هذه الوثيقة أنشطة التوعية الأمنية التي تم التخطيط لها وتقييمها للحد من المخاطر البشرية (مثل التصيد الاحتيالي والهندسة الاجتماعية) ورفع مستوى الوعي الأمني العام بالمنشأة.

---

## 🗓️ Core Activities / الأنشطة الرئيسية للحملة

### 1. Phishing Simulation / محاكاة التصيد الاحتيالي
- **Activity:** Sent mock phishing emails to test employee vigilance.
- **Result:** Identified high-risk departments and provided targeted retraining.
- ** إرسال رسائل بريد إلكتروني احتيالية وهمية لاختبار مدى يقظة الموظفين، وتحديد الأقسام الأكثر عرضة للمخاطر لإعادة تدريبهم.

### 2. Cybersecurity Awareness Infographics / تصاميم توعوية
- **Topics Covered:** Clean Desk Policy, Password Security, and Safe Remote Working.
- ** نشر منشورات توعوية رقمية تشمل: سياسة المكتب النظيف، أمان كلمات المرور، والعمل الآمن عن بُعد.

### 3. Interactive Training Module / وحدة تدريبية تفاعلية
- **Duration:** 15-minute mandatory security video covering basic cybersecurity hygiene.
 - ** فيديو تدريبي إلزامي مدته 15 دقيقة يغطي المبادئ الأساسية للأمن السيبراني لجميع الموظفين الجدد.
