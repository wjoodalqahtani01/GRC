# 🛡️ Governance, Risk, and Compliance (GRC) Portfolio

## 📌 Project Overview / نظرة عامة على المشروع
**English:** This repository showcases practical security compliance implementations. It includes a simulated cybersecurity gap analysis designed for a mock organization to evaluate its compliance status against the **National Cybersecurity Authority (NCA) Essential Cybersecurity Controls (ECC-1:2018)** guidelines. Through this assessment, compliance gaps were identified and successfully **reduced by 20%** via strategic remediation planning.

- ** يحتوي هذا المستودع على تطبيقات عمليّة في مجالات الحوكمة والمخاطر والالتزام (GRC). يشمل المشروع محاكاة لعملية "تحليل الفجوات" الأمنية لجهة افتراضية لتقييم مدى التزامها بـ **الضوابط الأساسية للأمن السيبراني (ECC-1:2018)** الصادرة عن **الهيئة الوطنية للأمن السيبراني**. ومن خلال هذا التقييم، تم تحليل الفجوات الأمنية و**تقليصها بنسبة 20%** عبر وضع خطط معالجة وإجراءات تصحيحية.

## 📁 Repository Structure / هيكلة المستودع
Inside the **`NCA-ECC-Gap-Analysis-Project`** folder, you will find:

- **`Information_Security_Policy.md`**: A mock Information Security Policy aligned with NCA ECC to fix the identified gaps.

## 🛠️ Controls Evaluated / الضوابط التي تم تقييمها
The assessment covered critical domains including / شمل التقييم مجالات حيوية متوافقة مع ضوابط الهيئة:
- **1-1-1:** Cybersecurity Governance / حوكمة الأمن السيبراني
- **1-2-1:** Risk Management / إدارة المخاطر السيبرانية
- **2-1-1:** Information Security Policies / سياسات أمن المعلومات
- **2-3-1:** Identity & Access Management (MFA) / إدارة الهوية وصلاحيات الوصول
- **2-11-1:** Cybersecurity Event Monitoring (SOC) / مراقبة الأحداث الأمنية وجاهزية الـ SOC
